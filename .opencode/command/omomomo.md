## What is DevSquad?

**DevSquad** is a powerful OpenCode plugin that transforms your AI agent into a full development team:

- 🤖 **Multi-Agent Orchestration**: Leader, Worker, Architect, Researcher, Scout, and more
- 🔧 **LSP Tools**: Full IDE capabilities for your agents - hover, goto definition, find references, rename, code actions
- 🔍 **AST-Grep**: Structural code search and replace across 25 languages
- 📚 **Built-in MCPs**: Context7 for docs, Exa for web search, grep.app for GitHub code search
- 🔄 **Background Agents**: Run multiple agents in parallel like a real dev team
- 🎯 **OpenCode Compatibility**: Your existing OpenCode config just works
- 🇨🇳 **Chinese Model Support**: Optimized for Qwen, MiniMax, GLM out of the box
